Cara Export File PCAP dengan Wireshark

  1. Buka aplikasi Wireshark dan mulai menangkap data jaringan dengan memilih interface yang sesuai.
  2. Setelah proses penangkapan selesai, klik menu File di bagian atas.
  3. Pilih opsi Save As.
  4. Di jendela yang muncul, beri nama file Anda dan pastikan formatnya .pcap.
  5. Klik tombol Save untuk menyimpan file.
