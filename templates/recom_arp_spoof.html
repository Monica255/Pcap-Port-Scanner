Tinjauan Umum ARP Anti-Spoofing

Apa Itu ARP?

ARP (Address Resolution Protocol) menyediakan komunikasi IP dalam domain siaran dengan memetakan alamat IP ke alamat MAC. Misalnya, host B ingin mengirim informasi ke host A tetapi tidak memiliki alamat MAC host A dalam cache ARP-nya. Dalam istilah ARP, host B adalah pengirim dan host A adalah target.

Bagaimana Serangan ARP Spoofing Terjadi?

Serangan spoofing ARP terjadi karena ARP mengizinkan balasan dari host meskipun permintaan ARP tidak diterima. Setelah serangan, semua lalu lintas dari perangkat yang diserang mengalir melalui komputer penyerang dan kemudian ke router, switch, atau host. Hal ini dikenal sebagai ARP cache poisoning.

Gambar: Contoh ARP Spoofing

Host A, B, dan C terhubung ke perangkat pada antarmuka A, B, dan C, yang berada pada subnet yang sama. Dengan memanfaatkan kelemahan ARP, host C dapat meracuni cache ARP host A dan B untuk menyadap lalu lintas di jaringan.

Cara Mencegah ARP Spoofing

  • Mengaktifkan ARP Anti-Spoofing: Semua paket ARP diperiksa dengan tabel pengikatan statis atau tabel ARP statis.
  • Mengonfigurasi Perlindungan Host: Mengikat alamat IP atau MAC ke port tertentu untuk mencegah manipulasi.
  • Mengonfigurasi Pemeriksaan Konsistensi Alamat MAC Sumber: Memastikan alamat MAC dalam paket ARP sesuai dengan tabel ARP.
  • Mengonfigurasi Gateway Anti-Spoofing: Mencegah host menyerang gateway dengan mengirim ARP palsu.
  • Mengonfigurasi Port Kepercayaan: Menentukan port tepercaya untuk meminimalkan pemeriksaan pada port yang terpercaya.

Serangan Flood ARP

Serangan flood ARP terjadi ketika paket ARP membanjiri CPU perangkat, memengaruhi kinerja jaringan. Untuk mencegahnya, Anda dapat mengaktifkan serangan anti-banjir ARP, mengatur ambang batas kecepatan aliran ARP, dan mengelola daftar alamat lubang hitam.

Langkah Pencegahan Tambahan

  • Mengonfigurasi Serangan Anti-Banjir: Mengidentifikasi dan membatasi aliran ARP berdasarkan ambang batas kecepatan.
  • Pemantauan ARP Snooping: Memastikan lalu lintas ARP diawasi untuk mencegah manipulasi data.
